Dentro da tag semantica "head", podemos acrescentar a Tag "style". Que resolverá o problema que o CSS inline possui, que é caso precisemos alterar a cor de todos os h1 da página, teríamos que alterar o atributo "style" em todos os h1.

Mas desta forma, utilizando a tag "style" dentro de "head", podemos alterar todos os h1. Para isso precisamos apenas selecionar os h1. Para saber como veja o código fonte desta página

Temos ainda, juntamente com a proriedade color, a propriedade "font-size", que configura o tamanho da fonte, nesse caso do h1, que é a tag selecionada.
