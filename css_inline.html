"style"

Este atributo é utilizado para estilizar elementos da nossa página HTML. Se você observar no código fonte dessa página, verá que esse atributo foi adicionado diretamente dentro da Tag h1

Porém essa não é uma boa prática pelo fato de que, se precisarmos alterar a cor de todos os h1, teriamos que adicionar esse atributo em cada h1 que houver dentro da página.
